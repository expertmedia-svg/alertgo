════════════════════════════════════════════════════════════════
  ARCHITECTURE
════════════════════════════════════════════════════════════════

AVANT (Twilio):
  Client SMS → Twilio Cloud ← Cloud → VPS
  Problème: Twilio n'a pas de numéros Burkina

APRÈS (Asterisk Hybride):
  SMS → Asterisk Local ↔ AMI (Port 5038) ↔ VPS
  Avantage: Contrôle complet, coûts réduits, couverture Burkina
